## Contributing

> [!NOTE]  
> Recommended Node.js version v18.16.0.

1. Fork the repo
2. Clone your fork
3. Install dependencies `npm install`
4. Run project `npm start` or `npm run ios`
5. Create a commit
6. Create a pull request
7. Wait for a review
8. Merge changes
