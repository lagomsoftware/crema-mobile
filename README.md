## Contributing

> [!NOTE]  
> Recommended Node.js version v18.16.0.

1. Fork the repo
2. Clone your fork
3. Install dependencies `npm install`
4. Run project `npm start` or `npm run ios`
5. Create a branch `git checkout -b [username]/[feature]`
6. Create a commit
7. Create a pull request
8. Wait for a review
9. Merge changes
